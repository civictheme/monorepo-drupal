# CivicTheme Starter Kit Drupal theme

Based on [CivicTheme](https://github.com/salsadigitalauorg/civictheme) Drupal theme.

## Compiling front-end assets.

    npm install
    npm run build

## Linting code

    npm run lint

    npm run lint:fix

## Starting a local Storybook instance

    npm run storybook
